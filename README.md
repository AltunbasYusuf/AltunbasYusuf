## 🌟 Featured Projects

### 🔹 Campus Navigation System
A comprehensive backend architecture designed for scalability. Built with continuous integration and delivery in mind, utilizing **Docker** for containerization and **Kubernetes** with **ArgoCD** for robust deployment and orchestration.

### 🔹 Social Photo Backend
A high-performance backend application built with **Node.js, Prisma, and PostgreSQL**. Designed using Clean Architecture principles, featuring a complete CI/CD pipeline and comprehensive API documentation via Swagger.

### 🔹 Vehicle Tracking System
A robust full-stack management system to track fuel costs, CO2 emissions, and service history. The backend is powered by **Java (Spring Boot)** and **MySQL**, seamlessly integrated with a cross-platform **Flutter** mobile application.
